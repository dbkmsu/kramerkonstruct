# KramerKonstruct Website Prototype

This is a GitHub Pages friendly static website prototype for KramerKonstruct.

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html` to the repository root.
3. In GitHub, open **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save and wait for GitHub Pages to publish the site.

## Contact form setup

The contact form uses a placeholder Formspree action:

`https://formspree.io/f/your-form-id`

Replace it with your actual Formspree endpoint to receive submissions.
